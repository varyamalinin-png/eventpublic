# ✅ Исправление DNS для Yandex Cloud Email API

## Проблема
Домен `mail-api.cloud.yandex.net` не разрешается через DNS на VM в Yandex Cloud.

## Решение
Добавлена запись в `/etc/hosts` для обхода проблемы с DNS:
```
178.154.131.0 mail-api.cloud.yandex.net
```

## Проверка
После добавления записи:
1. ✅ `getent hosts mail-api.cloud.yandex.net` должен вернуть IP
2. ✅ Приложение перезапущено через PM2
3. ✅ Теперь можно попробовать зарегистрироваться снова

## Важно
Это временное решение. Если IP адрес изменится, нужно будет обновить запись в `/etc/hosts`.

## Следующие шаги
1. Попробуйте зарегистрироваться в приложении снова
2. Проверьте почту - должен прийти код подтверждения
3. Если письмо не пришло, проверьте логи: `pm2 logs event-app`
