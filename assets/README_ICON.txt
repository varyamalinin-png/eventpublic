ИНСТРУКЦИЯ ПО ЗАГРУЗКЕ ЛОГОТИПА

1. Поместите ваш файл логотипа (солнце/звезда) в эту папку с именем: icon.png
2. Размер файла должен быть: 1024x1024 пикселей
3. Формат: PNG
4. После загрузки выполните: bash scripts/setup-app-icon.sh
5. Затем: npx expo prebuild

Ваш логотип будет автоматически использован для всех платформ!
