1. ОТМЕНИТЕ railway init (нажмите Ctrl+C)

2. Добавьте APP_BACKEND_BASE_URL в Railway:
   - Откройте https://railway.app
   - Проект: joyful-miracle → Сервис: eventpublic
   - Variables → + New Variable
   - Name: APP_BACKEND_BASE_URL
   - Value: https://eventpublic-production.up.railway.app

3. Проверьте Root Directory:
   - Settings → Source → Root Directory = "server"

4. Деплой:
   git add .
   git commit -m "Railway setup"
   git push

5. Проверка:
   - Откройте https://eventpublic-production.up.railway.app/health
   - Должен вернуться: {"status":"ok"}

✅ ВСЁ ГОТОВО!
