# Как установить приложение на конкретный телефон через Xcode

## Проблема
Приложение устанавливается на другой телефон, хотя провод подключен к нужному.

## Решение

### 1. Проверьте, какое устройство подключено

В Xcode:
1. Откройте Xcode
2. Window → Devices and Simulators (или `Cmd + Shift + 2`)
3. В левой панели вы увидите все подключенные устройства
4. Устройство с зеленой точкой - это то, что подключено сейчас

Или через терминал:
```bash
xcrun xctrace list devices
```

### 2. Выберите правильное устройство в Xcode

**В Xcode:**
1. Откройте проект: `ios/eventappnew.xcworkspace`
2. В верхней панели Xcode (рядом с кнопкой Play) выберите:
   - **Scheme**: `eventappnew`
   - **Destination**: Выберите нужный телефон из списка
   
3. Убедитесь, что выбран правильный телефон:
   - Нажмите на список устройств вверху
   - Выберите ваш телефон (тот, что подключен проводом)

### 3. Установка на выбранное устройство

**Вариант 1: Через Xcode GUI**
1. Выберите нужное устройство в верхней панели
2. Нажмите `Cmd + R` (Run) или кнопку Play
3. Приложение установится на выбранное устройство

**Вариант 2: Через терминал**
```bash
cd /Users/varya.malinina.2003mail.ru/event_app_new/client

# Посмотреть доступные устройства
xcrun xctrace list devices

# Установить на конкретное устройство (замените DEVICE_ID на ID вашего устройства)
npx expo run:ios --device DEVICE_ID

# Или просто выбрать из списка интерактивно
npx expo run:ios --device
```

### 4. Если устройство не появляется в списке

**Проверьте:**
1. Устройство разблокировано (экран включен)
2. На устройстве появился запрос "Доверять этому компьютеру?" - нажмите "Доверять"
3. Кабель поддерживает передачу данных (не только зарядку)
4. Переподключите кабель

**Вручную добавить устройство:**
1. Xcode → Window → Devices and Simulators
2. Если устройство не видно - отключите и подключите снова
3. На устройстве: Настройки → Основные → VPN и управление устройством → Доверие к компьютеру

### 5. Проверка, на какое устройство установится

**Перед установкой:**
- В Xcode в верхней панели должно быть написано имя вашего устройства
- Например: "iPhone 15 Pro" или "Вася iPhone"

**Если все еще устанавливается на неправильное устройство:**
1. Отключите WiFi на первом телефоне (чтобы он не мог получать приложение по беспроводной сети)
2. Отключите другие устройства от кабеля
3. Убедитесь, что в Xcode выбран правильный телефон

### 6. Альтернатива: Установка через TestFlight или Ad Hoc

Если проблемы продолжаются, можно использовать:
- TestFlight (для тестирования через App Store Connect)
- Ad Hoc distribution (для установки через файл .ipa)

## Быстрая команда для проверки подключенных устройств

```bash
# Показать все подключенные iOS устройства
xcrun xctrace list devices | grep -i "iphone\|ipad"
```

## Типичные проблемы

### Проблема: Устройство не доверяет компьютеру
**Решение:**
1. На iPhone: Настройки → Основные → VPN и управление устройством
2. Найдите ваш Mac в списке
3. Нажмите "Доверять этому компьютеру"

### Проблема: Xcode не видит устройство
**Решение:**
1. Отключите и подключите кабель снова
2. Перезапустите Xcode
3. Проверьте, что кабель поддерживает передачу данных

### Проблем: Приложение устанавливается на другой телефон по WiFi
**Решение:**
1. В Xcode: Preferences → Accounts → Download Manual Profiles
2. Или отключите WiFi на первом телефоне
3. Или выберите "Connect via Network" только для нужного устройства
